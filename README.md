# simplegrpc
Grpc golang and nodejs demo

```
// Server Side - golang
cd simplegrpc 
go run simpleserver.go


// Client side - nodejs
cd nodejs-client/
// do npm install
npm install  
// or yarn install 
yarn install 
node client_add.js 15 33
node client_sub.js 35 10
```
